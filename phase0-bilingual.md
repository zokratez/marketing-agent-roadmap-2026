Phase 0 – My Notes for Paco Peptide Marketing (Bilingual)

1. Workflow vs Agent: Workflow is a fixed checklist. Agent decides tools itself (ejemplo: agente que investiga competidores para Paco).

2. 4 Context Primitives: Write (notas de ideas), Select (saca datos de competidores), Compress (resumen corto), Isolate (agente separado para copia en español).

3. Harness vs Model: El código que escribimos (harness) hace que el modelo sea mucho mejor en tareas de marketing.

4. Orchestrator-worker: Lead agent planea la campaña y crea sub-agentes para investigación, copia y anuncios.

5. Failure modes: Demasiadas llamadas a herramientas (costo), olvida contexto, inventa hechos.

6. My goal: Build agents that generate bilingual campaigns for Paco Peptide (USA/Mexico Spanish speakers) to sell more products, and truly understand how marketing works. Keep tweaking campaigns in real time to know what trends are and what will be.
